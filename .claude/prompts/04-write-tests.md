# Write Tests

```
Напиши тесты для этого компонента.

Окружение: Create React App (Jest + @testing-library/react + @testing-library/jest-dom уже есть). Чистый JS.

Покрой:
- Рендер без ошибок
- Отображение переданных пропсов
- Пользовательские взаимодействия (клики, ввод) через userEvent
- Условный рендеринг / разные состояния
- Граничные случаи

Учти моки: react-i18next (t возвращает ключ), react-router (обернуть в MemoryRouter если нужно), Framer Motion при необходимости.
Используй getByRole/getByText, избегай тестирования деталей реализации.
```
