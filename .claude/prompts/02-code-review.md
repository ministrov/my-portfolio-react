# Code Review

```
Сделай ревью этого React-компонента (React 18, CRA, JS, Framer Motion, react-i18next).

Проверь:
- Хуки: правила хуков, массивы зависимостей, утечки в useEffect
- Производительность: лишние ре-рендеры, нужны ли memo/useMemo/useCallback, тяжёлые операции в рендере
- Доступность (a11y): семантика, alt, aria, фокус, навигация с клавиатуры
- i18n: захардкоженные строки
- Чистота: мёртвый код, магические числа, именование

Формат: таблица — Проблема | Серьёзность (critical/major/minor) | Файл/строка | Как исправить.
Не переписывай весь код, только точечные фрагменты для критичных правок.
```
